- `doodle1.svg` - E-Commerce Platform illustration (shopping cart doodle)
- `doodle2.svg` - Healthcare App illustration (heart and phone doodle)

These are colorful, hand-drawn style SVG illustrations that represent each project type. The component includes proper error handling that will display a placeholder if images are not found.

The SVG format provides:
- Scalable vector graphics that look crisp at any size
- Small file sizes for fast loading
- Customizable colors and styles
- Hand-drawn doodle aesthetic perfect for creative presentations
